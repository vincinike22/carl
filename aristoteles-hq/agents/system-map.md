# Agent System Map

## Hierarchy
- Aurelius: founder / final decision-maker
- Aristoteles: synthesis, prioritization, coordination, founder interface
- Leonardo: creative strategy
- Michelangelo: execution / builder
- Isaac: research
- Julius: communications / operations
- Steve: technical systems
- Carl: critical review / risk

## Communication pattern
- Specialists work in domain rooms
- Cross-functional issues go to `mission-control`
- Aristoteles owns escalation quality
- Founder-facing summaries stay compressed

## Founder protection rule
Aurelius should not receive uncompressed specialist chatter by default.
