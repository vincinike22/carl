# Welcome to Aristoteles HQ

Aristoteles HQ is a lean strategic operating system for Aurelius.

## What this is
- Aurelius is the founder and final decision-maker.
- Aristoteles is the central strategist and synthesis layer.
- Specialists contribute in focused domains:
  - Leonardo → creative
  - Michelangelo → build / execution
  - Isaac → research
  - Julius → communications / operations
  - Steve → technical systems
  - Carl → critical review / risk

## How to use this server
- Put raw work in the relevant specialist channel.
- Use `mission-control` for coordination and handoffs.
- Use `decisions-for-aurelius` for decision-ready summaries only.
- Use `daily-briefing` for concise daily synthesis.

## Noise policy
- Keep it useful.
- Default to concise.
- Do not narrate every thought.
- Bring evidence, options, risks, and recommendations.
- Escalate only what needs founder attention.

## Default workflow
1. Specialist develops input in their own lane.
2. Aristoteles synthesizes the important parts.
3. Aurelius receives compressed decision briefs and daily summaries.

This is not a roleplay server. It is an execution and thinking environment.
